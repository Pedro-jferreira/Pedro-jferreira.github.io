apresentação dos projetos finais de web 1

  1. Desenvolvimento do Site da Agência de Marketing "Digital Sphere Solutions"

    Visão Geral:

    O site Digital Sphere Solutions é um projeto web que utiliza HTML, CSS e JavaScript puro para criar uma experiência atraente e interativa. A seguir, apresento uma visão geral das principais funcionalidades do site:

    1. Página Inicial:

      A página inicial cumprimenta os visitantes com uma mensagem de boas-vindas, e destaca nossos principais serviços e recursos.

    2. Serviços:

      Uma seção dedicada a detalhar os serviços que oferecemos, incluindo marketing digital, publicidade online, otimização de mecanismos de busca (SEO) e gestão de redes sociais.

    3. Portfólio:

      Esta seção exibe exemplos de trabalhos anteriores, mostrando nosso histórico de sucesso em campanhas de marketing.

    4. Equipe:

      Aqui, os visitantes podem conhecer a equipe da [Nome da Agência], com informações sobre membros da equipe, suas funções e experiências.

    5. depoimentos:

      Implementamos um carrossel de depoimentos de clientes satisfeitos para construir confiança com os visitantes.

    6. Formulário de Contato:

      Um formulário de contato que permite que os visitantes entrem em contato conosco diretamente por meio do site.

    Tecnologias Utilizadas:

    • HTML, CSS e JavaScript:

      Utilizados para a criação da estrutura, estilo e interatividade do site.

    • Responsividade:

      O site é totalmente responsivo para garantir que funcione bem em dispositivos móveis e desktops.

  2. Desenvolvimento do Portfólio Pessoal

    Visão Geral:

    Meu portfólio pessoal é um projeto web que utiliza o framework Angular para criar uma experiência interativa e informativa. Aqui estão os principais elementos do projeto:

    1. Página Inicial:

      A página inicial apresenta uma breve introdução sobre mim e destaca minhas principais competências.

    2. Portfólio:

      Nesta seção, apresento uma lista de projetos em que trabalhei, incluindo links para os projetos no GitHub e uma breve descrição do meu papel em cada um deles.

    3. APIs Desenvolvidas:

      Criei páginas dedicadas para cada uma das APIs que desenvolvi. Cada página detalha o propósito da API, tecnologias utilizadas, documentação e exemplos de uso.

    4. Currículo:

      Disponibilizo meu currículo em formato PDF para download.

    5. Formulário de Contato:

      Um formulário de contato que permite que os visitantes entrem em contato comigo diretamente por meio do site.

    Tecnologias Utilizadas:

    • Angular:

      Utilizado como o framework principal para o desenvolvimento do site.

    • HTML, CSS, e TypeScript:

      Usados para criar a estrutura, estilização e funcionalidade do site.
